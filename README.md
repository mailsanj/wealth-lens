This is a hobby project that I started to learn vibe coding using Claude Code. This is a Wealth and Portfolio Management website app that supports creating and viewing portfolios (Stocks/ETFs/Bonds/Crypto/Real Estate etc.), building goals and scenarios using a variety of simulations (e.g., Monte Carlo, Historical etc.), and support for charting as well as support for exporting data to Excel. This supports Owner as well as Investment advisor type roles. The code uses Supabase for data storage as well as TwelveData API for real-time financial price updates of the portfolio. More information can be found under /docs.
